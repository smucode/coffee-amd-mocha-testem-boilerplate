coffee-amd-mocha-testem-boilerplate
===================================

Boilerplate project using CoffeeScript, RequireJS, AMD, Mocha, Chai, Jam and Testem.

```
# npm install -g jamjs
# npm install -g testem
# jam install
# testem
```
